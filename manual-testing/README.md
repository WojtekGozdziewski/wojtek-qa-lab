# Manual Testing Portfolio

## Overview

This section contains examples of manual testing activities performed on business applications.

The main focus areas are:
- business logic validation,
- data quality testing,
- functional testing,
- user experience verification,
- defect reporting.

## Tested Application

### Defalto CRM (Cloud Trial)

Testing activities were performed on a CRM system with focus on real-world business scenarios.

## Testing Approach

The testing process includes:

- exploratory testing to identify unexpected system behavior,
- analysis of business risks,
- creation of test scenarios,
- documentation of defects,
- evaluation of potential impact on users and business processes.
